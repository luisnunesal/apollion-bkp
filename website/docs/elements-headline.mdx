Heading são títulos de páginas e utilizados semanticamente como as tags HTML H1 até H6.

**Obs**: É possivel utilizar todas as propiedades do componente [Typography](/docs/elements-typography)

### Variantes

<Flex gap="large" mb="xl">
  <Flex gap="micro">
    <Headline variant="h1" text="Headline - h1" />
    <Headline variant="h2" text="Headline - h2" />
    <Headline variant="h3" text="Headline - h3" />
    <Headline variant="h4" text="Headline - h4" />
    <Headline variant="h5" text="Headline - h5" />
    <Headline variant="h6" text="Headline - h6" />

</Flex>

<Headline variant="h1" text="Headline - h1" />

## Headline API

| -------------- | --------------------------------------------------------------------------------------------------------------------------- | ------------- | --------- |
| children       | ReactNode                                                                                                                   |               |           |
| variant        | `h1` \| `h2` \| `h3` \| `h4` \| `h5` \| `h6`                                                                                | none          |           |
| align          | `start` \| `end` \| `left` \| `right` \| `center` \| `justify` \| `match-parent`                                            | none          |           |
