### Ações

Um alerta pode conter uma ação, como um botão de fechar ou desfazer.

<Flex mb="xl">
  <NotificationAction />
</Flex>

```jsx
<Notification
  closable
  type="block"
  variant="primary"
  icon={<Icon name="cog" />}
  title="I'm a Notification title :)"
  message="And I'm a text to complete notification XOXO"
  action={({ close }) => (
    <Button
      onClick={() => {
        // disparar alguma ação
        console.log("Ola Mundo");
        // por ultimo fechar o Notification
        close();
      }}
      size="small"
      text="Confirmar"
    />
  )}
/>
```

### Tipos

Há dois tipos principais de `Notification`:

- `block` (padrão)
- `page`
- [`toast`](#notificação-toast)

### Notificação Toast

Você pode usar o hook `useNotification` para exibir um notification do tipo toast.

<ToastExample />

```jsx
import { useNotification } from "@captalys-platform/core";

function Component() {
  const { showNotification } = useNotification();

  const successNotification = () => showNotification({
    title: "Super Titulo",
    message: "Messagem super criativa",
    variant: "success",
  });

  const warningNotification = () => showNotification({
    title: "Super Titulo",
    message: "Messagem super criativa",
    variant: "warning",
  });

  const dangerNotification = () => showNotification({
    title: "Super Titulo",
    message: "Messagem super criativa",
    variant: "danger",
  });

  const autoCloseNotification = () => showNotification({
    title: "Super Titulo",
    message: "Messagem super criativa",
    autoClose: 10,
  });

  return (
    <Flex flexDirection="row" gap="small">
      <Button onClick={displayNotification} text="Sucesso" />;
      <Button onClick={warningNotification} text="Warning" />;
      <Button onClick={dangerNotification} text="Danger" />;
      <Button onClick={autoCloseNotification} text="Auto Close" />;
    <Flex>
  )
}
```

### Notification API

| Propriedade  | Tipo                                            | Padrão    | Descrição                                                                                    |
| ------------ | ----------------------------------------------- | --------- | -------------------------------------------------------------------------------------------- |
| variant      | `primary` \| `success` \| `warning` \| `danger` | 'primary' | A cor da notificação.                                                                        |
| type         | `block` \| `page`                               | 'block'   |                                                                                              |
| pagePosition | `top` \| `bottom`                               | 'bottom'  | Quando o `type` for `page`, posiciona a notificação na parte inferior ou superior da página. |
| title        | string                                          | undefined |                                                                                              |
| message      | ReactNode                                       |           |                                                                                              |
| icon         | ReactNode                                       | null      |                                                                                              |
| button       | ReactNode                                       | null      |                                                                                              |
| linkName     | string                                          | undefined |                                                                                              |
| linkUrl      | string                                          | undefined |                                                                                              |
| closable     | boolean                                         | true      |                                                                                              |
| autoClose    | number                                          | undefined | Tempo em segundos para fechar a notificação automaticamente                                  |
