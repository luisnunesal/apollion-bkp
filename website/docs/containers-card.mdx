Um Card é um recipiente de conteúdo flexível e extensível. Ele inclui opções para cabeçalhos e rodapés, uma grande variedade de conteúdo, cores contextuais de fundo.

### Card com Mídia

O Card pode ter uma foto para ilustrar o conteudo, e além disso é possível mover os conteudos dentro dele na ordem que preferir através da prop `area`.

<Flex mb="xl">
  <Card
    width={300}
    title="Um titulo aqui"

    media={

        src="https://tecnoblog.net/wp-content/uploads/2019/07/pokemon-go-001-700x355.jpg"

### Card sem Mídia

Caso deseje um card sem imagem ele tambem pode ser feito apenas ocultando a prop `image`.

<Flex mb="xl">
  <Card
    width={300}
    title={
      <Flex flexDirection="row" alignItems="center" gap="small">
        <Icon name="roundedCheck" />
        <Typography variant="h5" text="Títutlo sem imagem!" />
      </Flex>
    }
  >
    <Typography
      color="primary"
      variant="p"
      text="Meu conteudo novo é só um link:"
    />
    <Link href="https://www.figma.com/">Acesse o Figma! :)</Link>
  </Card>
</Flex>

#### Ações

Apenas passe a prop `actionComponent` com qualquer componente.

<Flex mb="xl">
  <Card
    width={300}
    media={
      <Image
        cover
        src="https://tecnoblog.net/wp-content/uploads/2019/07/pokemon-go-001-700x355.jpg"
      />
    }
    actionComponent={
      <Flex flexDirection="row" alignItems="center" justifyContent="between">
        <Button size="small" color="danger" text="Deletar" variant="outlined" />
        <Button size="small" text="Confirmar" variant="contained" />
      </Flex>
    }
  >
    <Typography variant="p">Uhuul Botões de ação</Typography>
  </Card>
</Flex>

## Card API

| Propriedade     | Tipo                | Padrão    | Descrição                                                                       |
| --------------- | ------------------- | --------- | ------------------------------------------------------------------------------- |
| title           | string \| ReactNode | undefined | É o titulo principal do card. Permite passar componentes como icones e avatares |
| description     | string \| ReactNode | undefined | É o subtitulo do card. Permite passar outro component alem do texto.            |
| children        | ReactNode           | undefined | É o conteudo principal do card. Permite passar todo tipo de componente.         |
| media           | string              | undefined | Pode ser qualquer componente, preferencialmente Imagem, video.                  |
| actionComponent | ReactNode           | undefined | Passa-se os botões e funções como o usuario preferir                            |
