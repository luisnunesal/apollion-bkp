Utilize o `Paper` quando necessitar de sombras para indicar profundidade. `Paper` possui todas as props do `Flex`.

### Profundidade

`Paper` (`Flex` também) possuem 8 niveis de profundidade começando em -1 até 6.

<Flex mb="xl">
  <PaperDeepExample />
</Flex>

```jsx
<Paper bgColor="neutral.5" p="small">
  <Paper deep={-1} />
  <Paper deep={0} />
  <Paper deep={1} />
  <Paper deep={2} />
  <Paper deep={3} />
  <Paper deep={4} />
  <Paper deep={5} />
  <Paper deep={6} />
</Paper>
```

## Alterando cor das sombras.

`deepColor` altera a cor da sombra.

<Interactive>
  <Paper
    deep={select("deep", -1, [-1, 0, 1, 2, 3, 4, 5, 6])}
    deepColor={select("deepColor", "danger", colors)}
    p="small"
  >
    <Typography variant="h4" text="Lorem Ipsum" />
  </Paper>
</Interactive>
