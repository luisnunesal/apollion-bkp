## Spacing units table

| Alias            | Use                                         | REM unit | DP/PX Unit |
| ---------------- | ------------------------------------------- | -------- | ---------- |
| micro            | Espaçamento interno de agrupamentos         | .25      | 4          |
| xs               | Entrelinhas de agrupamentos                 | .5       | 8          |
| small            | Espaçamentos de inputs e de blocos internos | .75      | 12         |
| medium / Default | Espaçamento entre componentes externos      | 1        | 16         |
| large            | Espaçamento entre blocos de componentes     | 1.75     | 28         |
| xl               | Espaçamento entre blocos de conteúdo        | 2.75     | 44         |
| xxl              | Espaçamento entre seções de páginas         | 4.5      | 72         |
| giant            | Espaçamentos entre conteúdos diferentes     | 7.25     | 116        |
