Use a tipografia para apresentar seu design e conteúdo da forma mais clara e eficiente possível.

### Variantes

O uso de diferentes tamanhos e estilos de uma só vez pode estragar qualquer leiaute. Uma escala tipográfica tem um conjunto limitado de tipos de tamanhos que funcionam bem em conjunto com o leiaute de grade.

<Flex gap="large" mb="xl">
  <Flex gap="micro">
    <Typography variant="h1">Typography - h1</Typography>
    <Typography variant="h2">Typography - h2</Typography>
    <Typography variant="h3">Typography - h3</Typography>
    <Typography variant="h4">Typography - h4</Typography>
    <Typography variant="h5">Typography - h5</Typography>
    <Typography variant="h6">Typography - h6</Typography>
  </Flex>

  <Flex gap="micro">
    <Typography variant="lead">Typography - lead</Typography>
    <Typography variant="p">Typography - p</Typography>
    <Typography variant="meta">Typography - meta</Typography>
  </Flex>
</Flex>

```jsx
<Typography variant="h1">Typography - h1</Typography>
<Typography variant="h2">Typography - h2</Typography>
<Typography variant="h3">Typography - h3</Typography>
<Typography variant="h4">Typography - h4</Typography>
<Typography variant="h5">Typography - h5</Typography>
<Typography variant="h6">Typography - h6</Typography>

<Typography variant="lead">Typography - lead</Typography>
<Typography variant="p">Typography - p</Typography>
<Typography variant="meta">Typography - meta</Typography>
```

### Propriedades

É possivel alterar as seguintes `props` do `Typography`:

- `fontSize`
- `fontSize`
- `fontWeight`
- `lineHeight`
- `letterSpacing`
- `textTransform`
- `textDecoration`
- `fontStyle`
- `align`
- `color`

```jsx
<Typography fontSize="nano" />
<Typography fontWeight="bold" />
<Typography lineHeight="wild" />
<Typography letterSpacing="wild" />
<Typography textTransform="captalyze" />
<Typography fontStyle="code" />
<Typography color="danger" />
```

## Typography API

| Propriedade    | Tipo                                                                                                                        | Padrão        | Descrição |
| -------------- | --------------------------------------------------------------------------------------------------------------------------- | ------------- | --------- |
| children       | ReactNode                                                                                                                   |               |           |
| variant        | `h1` \| `h2` \| `h3` \| `h4` \| `h5` \| `h6` \| `p` \| `lead` \| `meta`                                                     | none          |           |
| align          | `start` \| `end` \| `left` \| `right` \| `center` \| `justify` \| `match-parent`                                            | none          |           |
| fontSize       | `nano` \| `micro` \| `small` \| `normal` \| `medium` \| `large` \| `xl` \| `xxl` \| `giant` \| `colossal` \| `gargantua` \| |               |           |
| fontWeight     | `light` \| `regular` \| `bold` \| `stronger`                                                                                |               |           |
| lineHeight     | `tight` \| `close` \| `regular` \| `wild`                                                                                   |               |           |
| letterSpacing  | `tight` \| `regular` \| `wild`                                                                                              |               |           |
| textTransform  | `uppercase` \| `lowercase` \| `captalyze` \| `none`                                                                         |               |           |
| textDecoration | `none` \| `overline` \| `underline` \| `line-through`                                                                       |               |           |
| fontStyle      | `normal` \| `code`                                                                                                          | `normal`      |           |
| wordBreak      | `all` \| `wrods` \| `normal`                                                                                                |               |           |
| truncate       | boolean                                                                                                                     |               |           |
| area           | String                                                                                                                      |               |           |
| color          | String                                                                                                                      | `neutral.180` |           |
| text           | String                                                                                                                      |               |           |
| htmlFor        | String                                                                                                                      |               |           |
