Quando falamos de títulos do tipo Display, estamos nos referindo aqueles utilizados em sua maioria para páginas com grandes CTAs e institucionais - normalmente se aplicam a landing pages e outras páginas de apresentação de conteúdo. São muito comuns em áreas de banner, páginas de login e de produtos.

**Obs**: É possivel utilizar todas as propiedades do componente [Typography](/docs/elements-typography)

### Variantes

## Headline API

| Propriedade    | Tipo                                                                                                                        | Padrão        | Descrição |
| -------------- | --------------------------------------------------------------------------------------------------------------------------- | ------------- | --------- |
| children       | ReactNode                                                                                                                   |               |           |
| variant        | `h1` \| `h2` \| `h3` \| `h4`                                                                                                | none          |           |
| align          | `start` \| `end` \| `left` \| `right` \| `center` \| `justify` \| `match-parent`                                            | none          |           |
| fontSize       | `nano` \| `micro` \| `small` \| `normal` \| `medium` \| `large` \| `xl` \| `xxl` \| `giant` \| `colossal` \| `gargantua` \| |               |           |
| fontWeight     | `light` \| `regular` \| `bold` \| `stronger`                                                                                |               |           |
| lineHeight     | `tight` \| `close` \| `regular` \| `wild`                                                                                   |               |           |
| letterSpacing  | `tight` \| `regular` \| `wild`                                                                                              |               |           |
| textTransform  | `uppercase` \| `lowercase` \| `captalyze` \| `none`                                                                         |               |           |
| textDecoration | `none` \| `overline` \| `underline` \| `line-through`                                                                       |               |           |
| fontStyle      | `normal` \| `code`                                                                                                          | `normal`      |           |
| area           | String                                                                                                                      |               |           |
| color          | String                                                                                                                      | `neutral.180` |           |
