`Flex` é o principal componente sobre o qual todos os outros componentes da IU do Apollion são construídos. Por padrão, ele renderiza um elemento div.

### Espaçamento

Com `Flex` é possivel criar espaçamento entre elementos e orienta-los em linha ou coluna, com as props `gap` e `direction`.

<Interactive>
  <Flex
    gap={select("Gap", "xl", [
      "micro",
      "xs",
      "small",
      "medium",
      "large",
      "xl",
      "xxl",
      "giant",
    ])}
    flexDirection={select("Direction", "row", [
      "row",
      "column",
      "row-reverse",
      "column-reverse",
    ])}
  >
    <Button variant="outlined" size="small" text="Cancelar" />
    <Button text="Comprar" size="small" />
  </Flex>
</Interactive>

Também é possivel controlar margin e padding usando atalhos

```jsx
<Flex p="micro" />       // padding
<Flex pt="xs" />         // padding-top
<Flex pb="small" />      // padding-bottom
<Flex pl="medium" />     // padding-left
<Flex pr="large" />      // padding-right
<Flex py="xl" />         // padding-top & padding-bottom
<Flex px="xxl" />        // padding-right

<Flex m="micro" />       // margin
<Flex mt="xs" />         // margin-top
<Flex mb="small" />      // margin-bottom
<Flex ml="medium" />     // margin-left
<Flex mr="large" />      // margin-right
<Flex my="xl" />         // margin-top & margin-bottom
<Flex mx="xxl" />        // margin-right
```

### Flex

Use `flex="initial` para permitir que um _**flex children**_ encolha, mas não aumente, levando em consideração seu tamanho inicial:

<Flex
  gap="large"
  borderRadius="md"
  color="neutral.0"
  bgColor="grayscale.5"
  p="large"
  mb="large"
>
  <Flex gap="small">
    <Typography text="Os itens não crescem quando há espaço extra" />
    <Flex flexDirection="row" gap="small">
      <Flex flex="initial" borderRadius="sm" p="small" bgColor="primary">
        <BaseText fontWeight="bold">Short</BaseText>
      </Flex>
      <Flex flex="initial" borderRadius="sm" p="small" bgColor="primary">
        <BaseText fontWeight="bold">Medium lenght</BaseText>
      </Flex>
    </Flex>
  </Flex>

  <Flex gap="small">
    <Typography text="Os itens diminuem, se possível, quando necessário" />
    <Flex flexDirection="row" gap="small">
      <Flex flex="initial" borderRadius="sm" p="small" bgColor="primary">
        <BaseText fontWeight="bold">Short</BaseText>
      </Flex>
      <Flex flex="initial" borderRadius="sm" p="small" bgColor="primary">
        <BaseText fontWeight="bold">Medium lenght</BaseText>
      </Flex>
      <Flex flex="initial" borderRadius="sm" p="small" bgColor="primary">
        <BaseText fontWeight="bold">
          Lorem ipsum dolor sit amet, consectetur adipisicing elit. Qui ad
          labore ipsam, aut rem quo repellat esse tempore id, quidem{" "}
        </BaseText>
      </Flex>
    </Flex>
  </Flex>
</Flex>

```jsx
<Flex flexDirection="row" gap="small">
  <Flex flex="initial">
    {/* Não vai crescer, mas vai encolher se necessário */}
  </Flex>
  <Flex flex="initial">
    {/* Não vai crescer, mas vai encolher se necessário */}
  </Flex>
  <Flex flex="initial">
    {/* Não vai crescer, mas vai encolher se necessário */}
  </Flex>
</Flex>
```

Use `flex="fluid"` para permitir que um _**flex children**_ aumente e diminua conforme necessário, ignorando seu tamanho inicial:

      </Flex>

Use `flex="none"` para evitar que um _**flex children**_ aumente ou diminua:

<Flex
  gap="large"
  borderRadius="md"
  color="neutral.0"
  bgColor="grayscale.5"
  p="large"
  mb="large"
>
  <Flex flexDirection="row" gap="small">
    <Flex flex="fluid" borderRadius="sm" p="small" bgColor="primary">
      <BaseText fontWeight="bold">
        Item que pode aumentar ou diminuir, se necessário
      </BaseText>
    </Flex>
    <Flex flex="none" borderRadius="sm" p="small" bgColor="primary">
      <BaseText fontWeight="bold">Item that cannot grow or shrink</BaseText>
    </Flex>
    <Flex flex="fluid" borderRadius="sm" p="small" bgColor="primary">
      <BaseText fontWeight="bold">
        Item que pode aumentar ou diminuir, se necessário
      </BaseText>
    </Flex>
  </Flex>
</Flex>


### Background

`Flex` aceita qualquer cor presente no tema como background, usando `bgColor`.

<Interactive hideCode>
  <FlexBackgroundExample color={select("Background", "danger", colors)} />
</Interactive>

```jsx
<Flex bgColor={color} height={100} justifyContent="center">
  <Typography
    contrast={color}
    text="lorem ipsum dolor sit amet"
    fontSize="large"
    fontWeight="bold"
  />
</Flex>
```

## Flex API

| gap            | number \| `SpacingInterface`                                      |           |                    |
| direction      | `row` \| `row-reverse` \| `column` \| `column-reverse`;           | `column`  |                    |
| flex           | `fluid` \| `initial` \| `auto` \| `none`                          |           |                    |
| justifyContent | `start` \| `end` \| `center` \| `between` \| `around` \| `evenly` | `start`   |                    |
| alignItems     | `start` \| `end` \| `center` \| `baseline` \| `stretch`           | `streach` |                    |
| wrap           | `wrap` \| `reverse` \| `nowrap`                                   |           |                    |
| order          | number \| `first` \| `last`                                       |           |                    |
| borderStyle    | string                                                            |           |                    |
| borderWidth    | string                                                            |           |                    |
| borderColor    | string                                                            |           |                    |
| borderRadius   | string                                                            |           |                    |
| bgColor        | string                                                            |           |                    |
| color          | string                                                            |           |                    |
| height         | number \| string                                                  |           |                    |
| maxHeight      | number \| string                                                  |           |                    |
| minHeight      | number \| string                                                  |           |                    |
| width          | number \| string                                                  |           |                    |
