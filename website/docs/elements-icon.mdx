Há três formas de usar o componente `Icon`:

- Usando ícones padrões do Apollion atraves da prop `name`.
- Ícones de customizados de terceiros com a prop `icon`.
- Ícones de customizados de terceiros com a prop `src`.

### Usando ícones do Apollion

basta passar o nome do icone `name`.

<Flex ml="small" mb="small">
  <Icon name="userConfig" size="xl" color="danger" />
</Flex>

```jsx
<Icon name="userConfig" size="xl" color="danger" />
```

### Usando ícones com `src`

Também é possivel utilizar link/url como fonte para o ícone.

**Obs**: É preferivel utilizar os metodos anteriores devido ao delay ao fazer o _fetch_.

<Flex ml="small" mb="small">
  <Icon
    src="https://raw.githubusercontent.com/FortAwesome/Font-Awesome/master/svgs/regular/clipboard.svg"
    size="xl"
    color="danger"
  />
</Flex>

```jsx
<Icon
  src="https://raw.githubusercontent.com/FortAwesome/Font-Awesome/master/svgs/regular/clipboard.svg"
  size="xl"
  color="danger"
/>
```

### Alterando o Tamanho

É possivel alterar o tamanho utilizando valores definidos, (`xs`, `small`, `xl`...) ou usando valores numericos.

<Flex ml="small" mb="small">
  <Icon name="userConfig" size={32} color="warning" />
</Flex>

```jsx
<Icon size={32} name="userConfig" color="warning" />
```

## Icon API

| iconProps   |                             | undefined      |           |
