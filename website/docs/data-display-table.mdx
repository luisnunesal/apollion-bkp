Tabelas são uma representação matricial, em linhas e colunas de informações de maneira
fácil e padronizada. No Apollion, retiramos cada importante caracteristica de uma Tabela
HTML e melhoramos para facilitar a vida dos usuarios e desenvolvedores.
As tabelas podem incluir:

- Uma visualização correspondente
- Navegação
- Ferramentas para consultar e manipular dados

### Elementos da Tabela

Nossa Tabela tem todos os elementos básicos necessários para criar a melhor tabela do mundo: `Table`, `TableHeader`, `TableHeaderCell`, `TableHeaderCheckboxCell`, `TableBody`, `TableRow`, `TableCell`, `TableCheckboxCell` e `TableFooter`.

Além disso você pode importar pelo _aliases_
abreviados mais similares ao **HTML**: `Thead`, `Th`, `Thc`, `Tbody`, `Tr`,`Thc`, `Td` e `Tfooter`.
