`CustomScroll` substitui a barra de rolagem padrão por uma mais simples e _clean_.

**Obs**: `CustomScroll` não suporta barra de rolagem horizontal.

```jsx
  <CustomScroll height={200}>
    {/* Seu conteúdo aqui */}
  <CustomScroll/>
);
```
